Preguntas Frecuentes // Soporte

¿La aplicación es gratuita?

Sí, puedes usar las funciones básicas completamente gratis. En el futuro podrían añadirse funciones premium.

¿Cómo invito a personas?

Desde la pestaña grupos podrás crear un grupo y añadir usuarios o usar un código de acceso.

¿Asignación específica?

Sí, puedes asignar tareas a miembros concretos para repartir las responsabilidades de forma justa.

¿Falta de cumplimiento?

La app envía recordatorios tácticos automáticos y muestra el progreso para fomentar la responsabilidad.

Sincronización Multi-dispositivo

Puedes acceder desde varios dispositivos y toda la información se sincroniza automáticamente en tiempo real.
